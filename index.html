Legacy fabric is a project based on the Fabric Project. Its two main priorities are to keep parity with upstream and to support as many legacy versions as possible. We currently support 1.8.9 and 1.12.2 (stable) and 1.7.10, 1.6.4 and 1.13.2 (experimental).

API

Legacy Fabric provides an API with common hooks and features to allow interoperability between mods.

Yarn

Legacy Fabric creates its own mappings for development. These are heavily inspired from Fabric's yarn.

Loader

Legacy Fabric for 1.8.9 uses a customized version of Fabric Loader to suit its needs to work with 1.8.9 .
