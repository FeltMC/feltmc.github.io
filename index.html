FeltMC aims to not only prove that QuiltMC is easy to best, but that it can all even be done on Fabric. Our goal is to run every Quilt feature on Fabric Loader, including Hashed Mojmap intermediaries, QSL, Quilt loader plugins, and even Quilt exclusive mods, all on the stability of Fabric Loader.

Fabricated-QSL

Felt provides a lib called Fabricated-QSL, which reimplements QSL on top of Fabric API for maximum mod compatibility, stability, and convenience..
